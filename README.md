# Food_Recogoniser
This project will help users get recipes for different south Indian food items along with its ingredients using Image recognitions. Users can scan the image of upload the photo of an Indian food item and and the project will return its ingridient list and recipie as an output.
# Howto run this project ?
1. Clone the repository: git clone "https://github.com/Parwathi-Jayaram/south_food"
2. Install all the dependencies listed in requirements.txt using pip install
3. Run the backend
4. Run the frontend using "streamlit run front.py"
